# NLCCS Website

Statische website voor [nlccs.nl](https://www.nlccs.nl) — het Nederlands Centrum voor Curriculumstudies.

## Stack

| Onderdeel | Keuze |
|---|---|
| Framework | [Astro v6](https://astro.build) — volledig statisch |
| CMS | [Sveltia CMS](https://sveltiacms.app) — git-based, `/admin` |
| Deploy | GitHub Actions → GitHub Pages |
